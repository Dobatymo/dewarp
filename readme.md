# Dewarp

Graphical user interface to perform a perspective transform ("dewarp") from a quadrilateral cropped from a picture to a rectangle.

## Requirements
- Python 3.6+

## Run
- `python3 dewarp.py` (Linux) or `py dewarp.py` (Windows).
